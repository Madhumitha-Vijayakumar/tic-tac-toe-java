Tic Tac Toe - Java Console Game

A simple 2-player Tic Tac Toe game built using Java for the console.  
Players take turns entering moves until there's a win or a draw.


Features

- 3x3 Tic Tac Toe grid
- Two-player turn-based game
- Input validation (only valid and empty cells can be used)
- Detects win and draw conditions
- Runs in any terminal or command prompt
